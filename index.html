Data Structure - Graph

Implementation of Graph Data Structure

A graph is a fundamental data structure in computer science that consists of a collection of nodes (or vertices) and edges connecting these nodes. It's a way to represent relationships between different entities.

To implement a graph data structure in JavaScript, you can use an adjacency list or an adjacency matrix. An adjacency list is a list of lists, where each list represents the neighbors of a vertex. An adjacency matrix is a square matrix, where each cell represents the weight of the edge between two vertices.
